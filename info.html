How Salaus works

Everything happens in your browser. No data is ever sent to any server.

Encryption standard

  • Algorithm: AES-256-GCM (authenticated encryption)
  • Key derivation: PBKDF2-SHA512 with 800 000 iterations
  • Salt: 32 random bytes (256-bit)
  • IV: 12 random bytes
  • Auth tag: 16 bytes (built-in)

For PC or Android

Install on Your PC or Android Device

Install Salaus as a Progressive Web App (PWA), on your PC or Android device and use it offline.

  1. Open in Browser
    Open this page in a supported browser (e.g., Chrome, Edge, Samsung Internet) over HTTPS.
  2. Install the App
    Click on the install button and then press install.

For iOS

Install on Your iOS Device

  1. Tap the Three Dots
    Tap the three dots next to the search bar in Safari.
  2. Select Share
    Tap Share from the menu.
  3. Tap More
    Select More (three dots) from the share options.
  4. Add to Home Screen
    Select Add to Home Screen (square with a + in the center).
  5. Enable Web App
    Check Open as Web App.

Note: This feature does not download the site but allows you to view it as an app. To use offline, open the shortcut, load the page, and then disconnect from the internet.
