# Salaus

## Site
https://pecoraroamal.github.io/Salaus/

**Client-side AES-256-GCM encryption** – 100% in your browser.  
No server. No tracking. No data leaves your device.

## Features
- AES-256-GCM with PBKDF2-SHA512 (800,000 iterations)
- Salt 32 bytes, IV 12 bytes
- Works with text and files (.txt, .json)
- Max file size: 0.5 MB
- Progressive Web App (PWA) – installable
- Fully offline
- Open source – MIT License


## How to use
Write or upload a .txt/.json file
2. Enter a strong password
3. Click **Encrypt** or **Decrypt**
4. Copy or download result

## Security
- Zero trust: everything happens in your browser
- Memory zeroing after use
- No cookies, no analytics
